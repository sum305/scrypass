# ScryPass

パスワードマネージャーとして機能するパスワードジェネレーター

## 使い方

各パラメーターの値を入力して生成ボタンを押すと、対応するパスワードが生成されます。生成後にページをブックマークすることで、パスフレーズ以外の入力を保存できます。

### パスフレーズ

秘密の文字列を入力してください。同じパスフレーズでも、サイトやソルトによって生成されるパスワードは変化します。

### サイト

認証先のドメイン名やユーザー名を、例えば `example.com` や `username@example.com` の様な形式で入力してください。

### ソルト

「ソルトを生成」をクリックすると、ソルト用に20桁のランダムな文字列が生成されます。

パスワードを更新するには再度生成を行うか、文字列に数字を付け加えて、更新の度にその数字を大きくしてください: `SALTFOOBAR-1` `SALTFOOBAR-2` ...

## 暗号強度

生成されるパスワードはscryptの導出鍵に基づいており、暗号強度はコストパラメーターによって増減します。既定値は 2^15 で、これはURLフラグメントの `cost=15` を書き換えることで変更できます。
